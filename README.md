# wechat-uos-makefile

统信版微信安装用 Makefile，改自 https://aur.archlinux.org/packages/wechat-uos/

Upstream: https://aur.archlinux.org/wechat-uos.git

## 安装
```sh
make install
```

## 清理下载文件
```sh
make clean
```

## 卸载
```sh
make uninstall
```
